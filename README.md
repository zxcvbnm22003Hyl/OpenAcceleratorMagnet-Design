# OpenAcceleratorMagnet-Design
## OpenAcceleratorMagnet-Design is an open-source project dedicated to the design of superconducting magnets for particle accelerators. Its aim is to provide a free and open platform for professionals, engineers, and students involved in this field. Currently, the project mainly focuses on the parametric design of cos(θ) type superconducting magnets (including CCT and DCT types), and supports the export of cond files recognizable by Opera for electromagnetic field simulation.
In the future, new features will be continuously added:
## 1. Parametric modeling and FEM analysis of geometric models for accelerator superconducting magnets, which can be directly integrated with software such as ANSYS and COMSOL.
## 2. A superconducting magnet design program developed based on Python, with an electromagnetic solver based on the line current model (completed).
## 3. New concepts for magnet design and optimization programs for accelerator superconducting magnets.
## 4. A program for calculating the AC loss of superconducting magnets.
