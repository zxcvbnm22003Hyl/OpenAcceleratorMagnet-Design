# OpenAcceleratorMagnet-Design
OpenAcceleratorMagnet-Design is an open-source project dedicated to the design of superconducting magnets for particle accelerators. It mainly focuses on the design of cos(θ) type superconducting magnets [CCT, DCT], and supports the export of cond files that can be recognized by Opera for electromagnetic field simulation.
